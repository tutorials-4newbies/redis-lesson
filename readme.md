# Redis introduction

-----

## Requirements
1. [Docker](https://docs.docker.com/engine/install/) engine installed
2. [Poetry](https://python-poetry.org/docs/#installation)


## Setup

1. Clone this repo.
2. Run and build the redis docker image:
    ```shell
    docker compose up
    ```
3. In a different shell install the dependence:
   ```shell
   poetry install
   ```
